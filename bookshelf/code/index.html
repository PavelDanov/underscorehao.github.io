Code (2nd edition)

Code shouldn’t need any introduction! It’s one of the best books on computers ever written — what more can I say?

If you have even the slightest interest in computers and how they work, this is the book. It’s an absolute gem.

Charles Petzold is a masterful storyteller who gradually guides readers through the principles and groundbreaking historical developments that led to the creation of the magnificent machine we call a computer today.

I’ve read the 1st edition three times over the years. When the 2nd edition came out, I bought it immediately to support Petzold, but I only recently read it in full. I’d say getting the 2nd edition, even if you already own the 1st, is a no-brainer.
