Plovdiv Game Jam 2025

I was a lecturer at this year’s Plovdiv Game Jam where I gave a presentation on Performance-Aware programming. The lecture was mostly an overview of some common ways that can help us when we want to write performant code.

Unfortunately, nowadays the majority of developers don’t think about such things at all. My time was quite limited, so I tried to show and explain as many starting points in this area as possible.

At the end of the day people that care about quality software also care about performance.
